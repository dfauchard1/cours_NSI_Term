    L'architecture des machines et les sytèmes d'exploitation⚓︎

    Dans ce cours nous utiliserons essentiellement le langage Python.

    Pourquoi ?

    Ce langage possède une syntaxe relativement simple qui nous permet de coder vite et bien. Les programmes fonctionnent rapidement et la mise au point est facilitée avec le mode interactif. Cet apprentissage avec Python vous donnera les bases essentielles de la programmation et sera facilement transposable sur un autre langage.

    L'environnement Python possède un atout de taille : de nombreuses librairies spécialisées permettent d'intégrer de nouvelles fonctionnalités en un claquement de doigts :

    • Le traitement des données : Panda, Bokeh
    • Le web : Django
    • Les mathématiques : Numpy, Scipy
    • L'intelligence artificielle : Keras, Tensorflow
    • Le jeu : Pygame
    • L'image : PIL, OpenCV
    • ...

    Python sur sa machine⚓︎

    Vérification de l'installation et de la version de Python installé sur votre ordinateur

    Il se peut qu'une version de Python soit déjà présente sur votre ordinateur. Suivez les indications ci-dessous pour le savoir

    1- Avec l'outil de recherche (la loupe) rechercher l'application cmd

    2- une fois l'application lancée, tapez la commande python --version

    Si python est présent, la commande répond en donnant la version existante sur l'ordinateur

    3- Pour connaître le répertoire d'installation, tapez la commande where python

    La commande répond en donnant tous les chemins trouvés

    Installer des librairies⚓︎

    Installer des librairies

    1- Avec l'outil de recherche (la loupe) rechercher l'application cmd

    2- une fois l'application lancée, tapez la commande python.exe -m pip install --upgrade pip pour mettre à jour l'application d'installation

    3- puis tapez la commande python.exe -m pip install pygame, dans cet exemple, il s'agit d'installer le package pygame

    Des outils pour développer⚓︎

    Les environnements de développement

    Pour créer des programmes vous pouvez utiliser un éditeur de texte standard, mais ce n'est pas le choix le plus pratique et le plus efficace.

    Un environnement de développement (IDE) fournit une interface adaptée aux différentes tâches du développeur.

    Plusieurs choix possibles :

    • des applications sur l'ordinateur, 2 essentiellement :
      • Thonny, un environnement léger qui permet un déboguage rapide
      • Visual Studio Code, un environnement plus professionnel
    • des applications en ligne :
      • Python Tutor, essentiellement pour apprendre les mécanismes principaux de la programmation
      • Basthon : un éditeur python en ligne
      • Replit : une application en ligne qui permet le travail en équipe sur le même fichier
    • Vous aurez aussi la possibilité de travailler directement en ligne sur ce site

    Un éditeur simple et basique, qui permet de faire du pas à pas facilement (pour déboguer ou comprendre un programme)

    1- pour le télécharger, RDV sur le site https://thonny.org

    2- télécharger la version de votre ordinateur (PC ou mac), pour l'installer sur votre machine

    3- cliquez sur le bouton 'install for users', puis pour les écrans suivants acceptez par défaut

    4- lors du premier lancement vous pouvez choisir la langue par défaut

    Thonny installe sa propre version de Python, ce qui oblige en particulier à installer les nouvelles librairies par l'intermédiaire de l'interface Thonny

    5- dans le menu Outils, avec la commande Gérer les paquets...

    6- Une recherche du module souhaité, puis son installation

    Il existe une option originale pour découvrir la programmation en Python grâce au logiciel en ligne Python Tutor. Cet outil permet de visualiser en live le déroulement d'un programme.
